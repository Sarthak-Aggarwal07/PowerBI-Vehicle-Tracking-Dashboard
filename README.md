# PowerBI-Vehicle-Tracking-Dashboard


## Project Overview

This project was developed using Power BI for monitoring vehicle movement and operational performance.

## Tools Used

- Power BI
- Dax
- Power Query
- SQL
- MySQL

## Features

- Vehicle Tracking Dashboard
- KPI Monitoring
- Route Analysis
- Trend Analysis
- Interactive Filters

## Skills Demonstrated

- Data Cleaning
- Data Modeling
- Table Relationships
- DAX Measures
- Data Visualization
- Dashboard Development
